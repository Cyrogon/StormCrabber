# StormCrabber

StormCrabber is a toll for Collecting your Workshop subscriptions from Steam and converting them into a `rom` style folder
that base Stormworks blocks and components use, this will allow you to join servers with matching components and skip the
workshop download step of joining, making servers with lots of component or map mods much faster to join.

Currently only has the ability to create a zipped file of then mods and backup your Stormworks vanilla `rom` folder.
